## Usage

Run the wallet application:

```bash
# Legacy version (single file)
python phantom_wallet.py

# New modular version (recommended)
python phantom_wallet_v2.py
```

### Quick Start

1. **Create a new wallet**: Generate a fresh wallet with new private keys
2. **Import existing wallet**: Restore your wallet using private key or seed phrase
3. **Check balance**: View your current cryptocurrency holdings
4. **Send transaction**: Transfer crypto to any address
5. **Receive crypto**: Get your receiving address to accept payments

## Security Features

- Secure random key generation
